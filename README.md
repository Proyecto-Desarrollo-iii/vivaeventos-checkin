# vivaeventos-checkin

Microservicio de validación de ingreso para VivaEventos.

## Descripción

Encargado de:
- Escaneo de QR en puertas
- Validación de boletas
- Control de ingreso (evitar doble entrada)
- Modo offline para señal irregular
